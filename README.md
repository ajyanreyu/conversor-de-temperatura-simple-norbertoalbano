# ETSII ULL Grado de Informatica

## Práctica: Conversor en JavaScript de grados Celsius a Farenheit y viceversa

### Que aprendemos

* HTML básico
* JavaScript básico
* Expresiones Regulares
* Un poquito de CSS
* Uso de Emmet (instale el plugin para su editor)
* Uso de Mocha y Chai
* Uso de Gulp
